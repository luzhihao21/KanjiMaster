## 🇺🇸 English

### 📝 Project Overview
**Kanji Master** is an iOS application designed to help learners master Japanese Kanji systematically. Built with **Flutter** and **Swift**, this app provides an interactive way to study Kanji based on structured textbook materials.

### 🚀 Key Features
- **Structured Learning**: Organized Kanji lists based on educational levels.
- **Cross-Platform Core**: Developed with Flutter for high performance and smooth UI.
- **iOS Optimized**: Fully compatible with iOS 18.0+ and tested on physical iPhone 14 hardware.

### 🛠 Tech Stack
- **Framework**: Flutter
- **Language**: Dart / Swift (Xcode Toolchain)
- **Deployment**: iOS 18.0 / 18.6.2
- **Version Control**: Git / GitHub

## 🇨🇳 简体中文

### 📝 项目简介
**“漢字達人” (Kanji Master)** 是一款专为日语学习者设计的 iOS 应用程序。本项目结合 **Flutter** 与 **Swift** 技术，基于专业教科书内容，为用户提供系统化、互动式的汉字学习体验。

### 🚀 核心功能
- **系统化学习**: 按照教学大纲整理汉字列表，循序渐进。
- **跨平台技术**: 使用 Flutter 构建，确保界面的流畅性与高性能。
- **iOS 深度优化**: 完美适配 iOS 18.0+ 系统，已在 iPhone 14 真机上完成部署与测试。

### 🛠 技术栈
- **框架**: Flutter
- **语言**: Dart / Swift (Xcode 工具链)
- **部署环境**: iOS 18.0 / 18.6.2
- **版本控制**: Git / GitHub
